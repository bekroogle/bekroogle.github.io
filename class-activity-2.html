Introduction

History

  • Ancient Greek mathematicians entertained themselves by doing a class of geometric puzzles called Compass-and-ruler constructions (a.k.a Classical constructions). Using only a compass and a straightedge, they would challenge themselves as well as others to construct certain geometric figures.

  • Some were exceedingly difficult (regular pentagon) while others were attempted for hundreds of years before finally proven to be impossible ( for more info).

C Spot Run: A web-based environment for developing and utilizing teaching languages for novice programming students.

This project comprises three parts:

  1. The programming language, c.run, designed for teaching novice programming students. Its compiler was build using PEG.js to run in the browser.
    • bekroogle/cspotrun-grammar
  2. A web-based editing environment for writing c.run programs.
    • www.cspotrun.org
  3. The web-based toolkit used to develop the c.run using the PEGjs parser generator.
    • http://pegedit.cspotrun.org

Relevance of Ancient Math

Ancient math is often found to be relevant in one of two ways:

  1. The seemingly outmoded system is still in use, particularly in specific or isolated areas.
  2. Techniques developed by the ancients can be applied to solve modern mathematical problems.

The Babylonian Number System

The Babylonian number system is a great example of one that is still used to this day. The Babylonians used sexigesimal, or base 60. We still use it in several fields:

  • Angular measurements (degrees)
  • Time (Hours, minutes, seconds)
  • This system has the benefit of being highly divisible: 60 can be cleanly divided by 2, 3, 4, 5, 6, 10, 12, 15, 20, and 30.

Microprocessors

Simple microprocessors aren't inherently capable of multiplication. Generally, they are equipped with only the most basic functions, such as:

  • Read/write from memory
  • Add/subtract
  • Doubling or halving a value ( for why this is)

 

How do we use these simple functions to implement multiplication?
